# TOON-Maker Frontend

A React-based user interface for the TOON-Maker converter.

## Features
- Input normal prompt
- Select language (English/French)
- View original, master, and TOON format outputs

## Setup

1. Install dependencies:
   ```sh
   npm install
   ```
2. Start the frontend server:
   ```sh
   node server.js
   ```

Frontend runs on `http://localhost:3000` and connects to backend at `http://localhost:5000/convert`.
